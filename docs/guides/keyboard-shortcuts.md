# Keyboard Shortcuts Reference

Complete keyboard shortcut reference for CardPlay, organized by context and board type.

## Global Shortcuts

Available in all boards and contexts:

| Shortcut | Action | Description |
|----------|--------|-------------|
| `Cmd/Ctrl + B` | Open Board Switcher | Switch between boards |
| `Cmd/Ctrl + K` | Open Command Palette | AI Composer command palette (AI boards only) |
| `Space` | Play/Pause | Toggle transport playback |
| `Enter` | Stop | Stop playback and return to start |
| `Esc` | Cancel/Close | Close modals, cancel operations |
| `Cmd/Ctrl + Z` | Undo | Undo last action |
| `Cmd/Ctrl + Shift + Z` | Redo | Redo last undone action |
| `Cmd/Ctrl + S` | Save Project | Save current project |
| `Cmd/Ctrl + Shift + S` | Save As | Save project with new name |
| `Cmd/Ctrl + O` | Open Project | Open existing project |
| `Cmd/Ctrl + N` | New Project | Create new project |

## Deck Navigation

| Shortcut | Action | Description |
|----------|--------|-------------|

| `Cmd/Ctrl + T` | New Deck Tab | Add new tab to deck container |
| `Tab` | Next Deck | Move focus to next deck |
| `Shift + Tab` | Previous Deck | Move focus to previous deck |

## Customization

**Note:** Keyboard shortcuts can be customized in **Settings â†’ Keyboard Shortcuts**. 

- All shortcuts support custom remapping
- Conflicts are detected and highlighted
- Import/export shortcut presets
- Reset to defaults available

## Platform Differences

- **Mac**: `Cmd` is used for most shortcuts
- **Windows/Linux**: `Ctrl` is used for most shortcuts
- Some shortcuts use platform-specific keys (e.g., `Option` vs `Alt`)

## Accessibility

- All shortcuts have alternative UI buttons
- Shortcuts respect system accessibility settings
- Screen reader announces shortcut actions
- Sticky keys and modifier locks supported
