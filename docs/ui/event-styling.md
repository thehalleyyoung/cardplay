# Event Styling System (J009)

The event styling system provides visual differentiation between events based on their origin: manual, generated, imported, or adapted.

## Features

- **Automatic origin detection** from event metadata
- **Visual indicators** with opacity and styling adjustments
- **High contrast support** with border-based differentiation
- **GPU-accelerated** CSS properties for performance
- **Accessible** color-independent indicators
